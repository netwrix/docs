# Configuring the Monitoring Plan

This topic explains how to configure a monitoring plan.

## Overview

Follow the steps below to configure the monitoring plan.

To use the monitoring plan feature, ensure your system meets the prerequisites.

## Steps to Configure

1. Click **Settings** in the navigation bar.
2. Select the appropriate monitoring template.
3. Click **Save**.
4. Changes take effect after you restart the service.

## Important Information

This feature helps administrators monitor their systems. If you have any questions, don't hesitate to reach out to the support team.

The monitoring plan allows you to configure alerts, set thresholds, and manage notifications. Test all changes in a non-production environment before deploying to production.

:::note
You must have administrative privileges to access this feature. Contact your system administrator if you do not have the necessary permissions.
:::
