# Setting Preferred Forms

You can set the preferred form to use for specific tasks.

1. Open **Customization** > **Lists, Records, & Fields** > **Record Types**
2. Select a form from the list. For this example, select **Change Request**.

    ![Setting the preferred form.](/images/platgovnetsuite/customizations/preferred_form1.webp)

3. Open the **Forms** tab.

    ![Setting the preferred form.](/images/platgovnetsuite/customizations/preferred_form2.webp)

4. Click the **Preferred** radio button to make the new **ITGC**Â Change Request form the preferred
   form.
5. Click **Save**.
