## Installation Steps:

The software is installed by the administrator on the server.  After the installation is completed, the system will be configured by opening the admin panel.

1. First, login to the admin console
2. Click on the **Settings** tab
3. The checkbox(es) should be selected by you for the feature(s) you want
4. As shown below, the dropdown menu provides the ability to select your region
5. Follow the steps to configure the network settings
6. Click on **Save** to apply the changes

> **NOTE:** Note that the settings were recently updated. Currently, as of this writing, the setup process is not what it was before — it is streamlined.
