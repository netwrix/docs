# Delete a User

Using this API, you can delete a single user.

## Endpoint

**https://machinename:4443/GroupIDDataService/api/IdentityStores/{identitystoreId}/Users/{userIdentity}**

## HTTP Method

**DELETE**

```
    "message": null,
    "status": 0
}
```
