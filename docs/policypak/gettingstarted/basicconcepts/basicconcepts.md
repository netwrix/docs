# Introduction and Basic Concepts

Thank you for trying Netwrix PolicyPak Software. This manual will
discuss the following PolicyPak policies:

| Component                      | Ability                                                                                                                                                                                                                                                                      |
| ------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [Application Settings Manager](/docs/policypak/components/applicationsettingsmanager/)   | Enforces and locks down your application settings across your network.                                                                                                                                                                                                       |
| [Least Privilege Manager](/docs/policypak/components/leastprivilegemanager/overview)        | Eliminates local admin rights instantly and stops crypto-malware and unknown threats immediately.                                                                                                                                                                            |
| [Browser Router](/docs/policypak/components/browserrouter/overview)                 | Doesn't leave browsing to chance! It manages multi-browser environments by mapping specific websites to specific browsers.                                                                                                                                                   |
| [Java Enterprise Rules Manager](/docs/policypak/components/javaenterpriserules/overview)  | Maps which websites should use which specific versions of Java.                                                                                                                                                                                                              |
| [Admin Templates Manager](/docs/policypak/components/admintemplatesmanager/)        | Enables you to place Microsoft Admin Template settings into Collections, and targets conditions under which the settings should apply. The MMC snap-in also exports Group Policy settings to be deployed using SCCM (for example) and through PolicyPak Cloud. |
| [File Associations Manager](/docs/policypak/components/fileassociationsmanager/overview)      | Maps file extensions to applications and maps protocol extensions to applications.                                                                                                                                                                                           |
| [Preferences Manager](/docs/policypak/components/preferencesmanager/overview)            | Exports Group Policy Preferences settings to be deployed using SCCM (for example) and through PolicyPak Cloud or your own MDM service.                                                                                                                         |
| [Security Settings Manager](/docs/policypak/components/securitysettingsmanager/overview)      | Exports Group Policy Security settings to be deployed using SCCM (for example) and through PolicyPak Cloud or your own MDM service.                                                                                                                            |
| [Start Screen & Taskbar Manager](/docs/policypak/components/startscreenandtaskbar/) | Maps specific apps to Windows 10 Start Screen groups and pins items to the Windows 10 Taskbar                                                                                                                                                                                |
| [Scripts & Triggers Manager](/docs/policypak/components/scriptstriggers/overview)     | Run PowerShell, JavaScript, Visual Basic and Batch scripts and target your users and computers using SCCM (for example) and through PolicyPak Cloud or your own MDM service.                                                                                   |
| [Remote Work Delivery Manager](/docs/policypak/components/remoteworkdeliverymanager/overview)   | Gets apps and other files to on-prem or cloud endpoints.                                                                                                                                                                                                                     |
| [Feature Manager for Windows](/docs/policypak/components/featuremanager/)    | Select and deselect Windows features that best fit your needs in an unobtrusive manner.                                                                                                                                                                                      |
| [RDP Manager](/docs/policypak/components/remotedesktopprotocolmanager/overview)                    | Easily create, manage, and deploy RDP files for remote users.                                                                                                                                                                                                                |
| [Network Security Manager](/docs/policypak/components/networksecuritymanager/)       | Ensure specific processes utilize specific network addresses and protocols.                                                                                                                                                                                                  |
| [Software Package Manager](/docs/policypak/components/softwarepackage/)       | Quickly add or remove AppX packages from the Microsoft Store.                                                                                                                                                                                                                |

PolicyPak has one goal for your users' applications: to ensure that we are Securing
YOUR Standards™.
