# Create Subscriptions

Subscriptions let you receive Netwrix Auditor overview and SSRS reports, search results, behavior anomaly dashboards, and risk assessment overviews on a scheduled basis — delivered by email or saved to a file share. Use subscriptions to stay informed about activity in your environment without logging into the product.

## Prerequisites

You must have the Global administrator or Global reviewer role in the product. For details, see [Role-Based Access and Delegation](/docs/auditor/10.8/admin/monitoringplans/delegation.md).

## Procedure

1. Do one of the following depending on subscription type:

    - **Subscribe to an Overview or SSRS report** — An Overview or SSRS report subscription can be created in one of the following ways:
        - From the Reports page: on the main Auditor page, navigate to Reports. Specify the report you want to subscribe to and click **Subscribe**.
        - From the report view: open the report and click **Subscribe**.
    - **Subscribe to Search** — A Search subscription can be created in one of the following ways:
        - From the search results page (a search must have been performed first): navigate to Tools and select **Subscribe**.
        - From a predefined or user-saved search report: on the main Auditor page, navigate to Reports. Specify the Search report you want to subscribe to and click **Subscribe**.
    - **Subscribe to Behavior Anomalies** — On the main Auditor page, navigate to Behavior anomalies, then in the dashboard window click **Subscribe**.
    - **Subscribe to Risks** — On the main Auditor page, navigate to Risk assessment and in the dashboard window click **Subscribe**.
