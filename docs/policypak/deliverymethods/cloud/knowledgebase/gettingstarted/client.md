# When must I use the PolicyPakCloud Client installer versus the on-prem PolicyPak CSE?

The Netwrix PolicyPak Cloud Client is the software you download
to enable a machine to join your instance of PolicyPak Cloud, claim a license, and
download policies.

The PolicyPak Cloud Client will also download the PolicyPak Client Side
Extension to process of the policies.

When you install the PolicyPak cloud client, the CSE is downloaded from the cloud
service automatically at the same time.

To keep the PolicyPak Cloud Client up to date, as well as the PolicyPak
Client Side Extension, you need to be proactive.

Please see this article for keeping things proactive:

[https://www.policypak.com/resources/pp-blog/using-rings-to-test-and-update-the-policypak-client-side-extension-and-how-to-stay-supported/](https://www.policypak.com/resources/pp-blog/using-rings-to-test-and-update-the-policypak-client-side-extension-and-how-to-stay-supported/)
