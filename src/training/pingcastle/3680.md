## 3680 Introduction to <NPC />

Recommended prerequisite: 2680 <NPC /> – Concepts & Architecture

The Introduction to <NPC /> course provides a comprehensive overview of identity security assessment workflows, covering standalone analysis, enterprise deployment, web interface navigation, and automated monitoring of security posture over time. This course is comprised of four modules:

* Standalone Installation and Assessment
* Enterprise Installation and Configuration
* Navigating and Operating the Web Interface
* Automation and Validation

Estimated length: 1 hour 25 minutes
