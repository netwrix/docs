# Getting Started

Netwrix PolicyPak Cloud is a way to deliver the following items:

- Any PolicyPak directive you are licensed for, such as PolicyPak Least
  Privilege Manager, PolicyPak Browser Router, etc.
- Any Microsoft directive you are licensed for, such as Microsoft ADMX settings, Microsoft Group
  Policy Preferences settings, and Microsoft Group Policy Security settings.

This document is a QuickStart Guide for PolicyPak Cloud and our full User Guide for
PolicyPak Cloud. It will help you understand PolicyPak Cloud and how the
PolicyPak components work with it.

:::note
For more details on any of the PolicyPak components themselves, see the
related manual for that component.
:::


Getting started with PolicyPak Cloud requires you to talk with PolicyPak
Sales. PolicyPak Sales will set up your PolicyPak Cloud trial account,
which is typically set up as follows:

- 10 licenses for PolicyPak Cloud, valid for 30 days
- Enablement of all components
- Required computer check-in to PolicyPak Cloud every 14 days (or the computer's
  license is released back into the pool)

After you're done testing and you're ready to get a pool of licenses, which are good for a year,
contact Netwrixsupport to obtain a license.
