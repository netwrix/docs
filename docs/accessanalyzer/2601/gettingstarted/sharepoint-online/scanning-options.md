# Available Scanning Options

| Scan Option | Description | Available Configurations |
| --- | --- | --- |
| **SharePoint Online Access** | Scans SharePoint Online sites for permissions and access controls. The first scan runs in full; subsequent scans run differentially, collecting only changes since the last run. | Site URLs (optional — leave empty to scan all sites) |

## Scan Configuration

**SharePoint Online Access**

- **Site URLs** — Leave empty to scan all sites in the tenant, or enter specific site URLs to limit the scan to a subset of your SharePoint Online environment.
