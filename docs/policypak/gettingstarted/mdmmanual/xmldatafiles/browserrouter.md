# XML Data Files from PolicyPak Browser Router

PolicyPak Browser Router settings can be exported as an XML file.
Right-click` Computer Configuration | PolicyPak | Browser Router` or
`User Configuration | PolicyPak | Browser Router`, and pick the collection you wish to export, as
shown in Figure 13. For full details on the PolicyPak Browser Router, see the
[Browser Router](/docs/policypak/components/browserrouter/overview.md).

![deploying_policypak_directives_12](/images/policypak/mdm/xmldatafiles/deploying_endpointpolicymanager_directives_12.webp)

Figure 13. Exporting a collection as an XML file via PolicyPak Browser Router.
