# Configure Alert Notifications

This document helps users configure alert notifications for their monitoring plan.

## Overview

We believe that with Access Analyzer, you can simply and easily configure alert notifications in order to monitor your environment. Basically, the setup process is effortless for any administrator.

Without Access Analyzer, organizations cannot properly manage their alerts. It is recommended that you utilize this feature to ensure compliance.

Please note that alert notifications are not just emails, alert notifications are a comprehensive communication system that keeps your team informed.

## Steps to Configure

1. First of all, it is important to navigate to the **Alerts** page
2. The notification template should be selected by the user
3. Please be advised that you can simply and easily customize thresholds
4. The configuration can be saved by clicking **Save**

## Important Information

Note that without proper alert configuration, administrators have no visibility into security events. We strongly recommend that all organizations utilize this feature.
