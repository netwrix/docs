# Select Exchange Objects from Active Directory Window

The Select Exchange Objects from Active Directory window provides a list of Active Directory users
that have mailboxes configured, to either include or exclude. Selected objects are added to the
filter from where you clicked the **Add** (+) button to open this window.

![Select Exchange Objects from Active Directory Window](/images/threatprevention/8.0/admin/policies/eventtype/window/selectactivedirectory/exchangeobjectsfromad.webp)

- Select an Agent from the drop-down menu and click **Connect**.
- Navigate through the domain tree and check the desired item(s) in the **Results** pane on the
  right.
- Then click **OK** to close the window.

See the [Selection Windows](/docs/threatprevention/8.0/admin/policies/configuration/eventtype/window/overview.md) topic for additional information.

The selection is displayed in the appropriate box of the Exchange Mailbox Objects and Containers
filter.
