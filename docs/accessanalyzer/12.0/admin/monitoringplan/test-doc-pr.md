# Configuring the monitoring plan

This document is designed to help users in the process of configuring their monitoring plan. It was written by the team to provide assistance.

## Overview

Use the monitoring plan to monitor various aspects of the system. Follow the steps below to configure the monitoring plan.

To use the monitoring plan, ensure your system meets the prerequisites.

## Steps to Configure

1. Click **Settings** in the navigation bar.
2. Select the appropriate monitoring template.
3. Click **Save**.
4. Restart the service for changes to take effect.

## Important Information

This feature helps administrators monitor their systems. If you have any questions, contact Netwrix support.

The monitoring plan allows you to configure alerts, set thresholds, and manage notifications.

:::warning
Test all changes in a non-production environment before deploying to production.
:::

:::note
You must have administrative privileges to access this feature. Contact your system administrator if you do not have the necessary permissions.
:::
