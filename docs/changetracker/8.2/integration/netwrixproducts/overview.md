# Netwrix Products

Netwrix Change Tracker integrates with the following Netwrix products:

- [Netwrix Auditor Integration](/docs/changetracker/8.2/integration/netwrixproducts/netwrixauditor.md) – Netwrix Change Tracker can be configured to
  send event data collected by Change Tracker to Netwrix Auditor.
- [Netwrix Activity Monitor Integration](/docs/changetracker/8.2/integration/netwrixproducts/activitymonitor.md) – The Gen 7 Agent can use
  Netwrix Activity Monitor as the data source for file change attribution on Windows, as an
  alternative to the built-in kernel mini-filter driver.
