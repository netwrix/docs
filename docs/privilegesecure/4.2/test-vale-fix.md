# Getting Started with Activity Sessions.

We recommend that you click on **Settings** to begin configuring activity sessions. Please note that once you have hit **Save**, the system allows you to use activity sessions across your environment.

## Prerequisites

It is important to note that you must have administrative privileges before proceeding. Currently, only administrators can configure activity sessions.

## Configure an Activity Session

Follow these steps:

1. Navigate to **Admin > Activity Sessions**.
2. Click on **New Session** to create a new activity session.
3. Type a name into the **Session Name** field  and select a session type.
4. The session type allows you to define the scope of monitoring, manage permissions and control access levels.
5. We wish to point out that session names must be unique.
6. Hit **Save** to apply your changes.

## Monitor Activity

Once you have configured activity sessions, you can monitor them from the dashboard. The dashboard allows you to view active sessions, review logs and manage session settings.
