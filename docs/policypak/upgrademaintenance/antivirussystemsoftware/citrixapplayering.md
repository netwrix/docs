# I want to use PolicyPakwith Citrix App Layering (aka Unidesk). At which layer should I implement the PolicyPak Client Side Extension?

Citrix App Layering lets you add packages at the OS, PLATFORM or APP LAYER.

While it ispossible that Netwrix PolicyPak should work at any
layer, we recommend the OS layer since PolicyPak acts as part of the operating system
and is tightly integrated with Group Policy.
