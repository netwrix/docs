# Baseline

Being the victim of a cyber attack can be scary, expensive, and potentially business-crippling.

So how do you prevent a cyber attack? Start with making security a priority for all IT operations,
and the first place to start is by making systems as hacker proof as possible:

Gold Build Standard? Corporate Build or Hardened Build? Controlled Image? Baseline Configuration?
You will find the need for a Gold Build Standard in all compliance frameworks (for example, NIST
800-53 CM-2 and CM-3, CIS Control 5.2, PCI DSS Requirement 2 and especially NERC CIP 007-3 and
010-3) as a means of guaranteeing security. Without a consistent build how else can you expect
security to be maximized?
