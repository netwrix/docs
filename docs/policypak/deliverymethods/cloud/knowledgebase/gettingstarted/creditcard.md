# How do I start credit card billing with PolicyPak SaaS Edition?

To start Netwrix PolicyPak SaaS / Cloud billing, you need to go
the PolicyPak Portal (not the Cloud Service.)

Then when you're there, click on **SaaS Billing**, then **Start Subscription**.

Follow the directions after that.
