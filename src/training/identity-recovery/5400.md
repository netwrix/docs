## 5400 <NIR /> – Demo the Basic Use Cases

Recommended prerequisite: 3400 Introduction to <NIR />

The <NIR /> – Demo the Basic Use Cases course provides you with the ability to demonstrate the two basic use case for this application, formerly Netwrix Recovery for Active Directory:

* Provide Object and Attribute Level Recovery
* Forest Level Recovery

When you complete this course, you will understand the scenario and demonstration talking points for each use case.
