# Licensing Details

## Overview

The purpose of this article is to explain how licensing statistics can be found within Privilege
Secure. This doesn't override specifics within the software agreement.

**NOTE:** Access Risk needs to be enabled for the license calculation to generate.

## Main Document

The licensing statistics can be found in the left hand menu, under the Configure option:

![Screen_Shot_2022-06-01_at_8.50.L00_AM.webp](/images/privilegesecure/4.2/discovery/admin/reporting/6472622083095_screen_shot_2022-06-01_at_8.50.00_am.webp)

This opens to show the license utilization over the prior 90 days

![Screen_Shot_2022-06-01_at_8.49.L50_AM.webp](/images/privilegesecure/4.2/discovery/admin/reporting/6472622083095_screen_shot_2022-06-01_at_8.49.50_am.webp)

This is broken down by operating system and, in the case of windows, by server and desktop/laptop.
The time frame can be adjusted to the prior month or week if needed.

The License Use Details link will give further insight showing median and daily information:

![Screen_Shot_2022-06-03_at_1.54.L04_PM.webp](/images/privilegesecure/4.2/discovery/admin/reporting/6472622083095_screen_shot_2022-06-03_at_1.54.04_pm.webp)
