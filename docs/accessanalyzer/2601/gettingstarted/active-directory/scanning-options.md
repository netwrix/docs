# Available Scanning Options

| Scan Option | Description | Available Configurations |
| --- | --- | --- |
| **Active Directory Inventory** | Scans users, groups, and group memberships from all domain controllers in the source group. The first scan runs in full; subsequent scans run differentially, collecting only changes since the last run. | None |
