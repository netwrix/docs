# How to Set Up Your First Scan

After installing the product, follow the steps below to configure your first scan. To get started, log in to the console and ensure your credentials are configured.

## Getting Started

With Netwrix Auditor, you can monitor changes across your environment. It's a complete auditing solution, not just a reporting tool. The product lets you track changes and uses advanced algorithms to detect anomalies.

To configure a new scan, click **New Scan**. When the dialog opens, select the scan type from the dropdown menu.

:::note
Some scan types require additional configuration.
:::

## Configuring the Scan

Follow the steps below to set up your scan:

1. Click **Settings** in the navigation panel.
2. Select the monitoring plan that you want to configure.
3. Enter the target paths to define the scan scope.
4. Review the settings, then click **Run** to start the scan.
5. The system generates the report.
6. See [the configuration reference](https://docs.netwrix.com) and [additional resources](https://example.com).

Ensure you have the correct permissions before proceeding. The items you select determine the scan results. For example, selecting a broader scope takes longer because it scans more objects.

This feature supports Windows servers. Linux support is planned for a future release. For more details, refer to the section above.
