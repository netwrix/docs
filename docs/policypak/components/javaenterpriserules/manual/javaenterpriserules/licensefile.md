# Checking the License File

PolicyPak Java Rules Manager must be licensed like every other PolicyPak
component. If you are using PolicyPak, then you need a PolicyPak Java
Rules Manager license file, which can be imported into a Group Policy Object (GPO), as shown in
Figure 29.

![troubleshooting_policypak](/images/policypak/troubleshooting/javaenterpriserules/troubleshooting_endpointpolicymanager.webp)

Figure 29. PolicyPak Java Rules Manager must be licensed like every other PolicyPak component.
