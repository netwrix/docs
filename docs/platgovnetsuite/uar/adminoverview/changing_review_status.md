# Changing a Single Review Status

The status of a Single Review and be changed from **Complete** to **Not Started**. An Administrator
or UARÂ Admin role is required to make the status change.

1. Note the Name of the review to be changed. This example uses _0001_Membership_SOD Edit Role_.

    ![Note the review name](/images/platgovnetsuite/uar/uar_admin/uar_change_status1.webp)

2. Open **Customization** > **List, Records, & Field** > **Record Types**
3. Locate **Review** in the list.

    ![Locate Review in the Edit column](/images/platgovnetsuite/uar/uar_admin/uar_change_status2.webp)

4. Click **List** in the **Review** row.

    ![List displayed for Review](/images/platgovnetsuite/uar/uar_admin/uar_change_status3.webp)

5. Click **Edit** in the row of the review to be changed. _0001_Membership_SOD Edit Role_ for this
   example.

    ![Edit the review](/images/platgovnetsuite/uar/uar_admin/uar_change_status4.webp)

6. Change **Status** from **Complete** to **Not Started**.
7. Click **Save**.
