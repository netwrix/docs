# Configure the Monitoring Plan

## Overview

You can configure the monitoring plan to monitor various aspects of the system.

To use the monitoring plan feature, ensure your system meets the prerequisites.

## Steps to Configure

1. Click **Settings** in the navigation bar.
2. Select the appropriate monitoring template.
3. Click **Save**.
4. Restart the service for the changes to take effect.

## Important Information

The monitoring plan allows you to configure alerts, set thresholds, and manage notifications. For questions, contact Netwrix support.

:::warning
Test all changes in a non-production environment before deploying to production.
:::

:::note
You must have administrative privileges to access this feature. Contact your system administrator if you do not have the necessary permissions.
:::
