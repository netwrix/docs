# Event Logs

The PolicyPak GPCR client (admin console) logs warnings and errors to the Windows
Event Viewer in the application log. Because a lot of data could be in the log, use the filters
shown in Figure 76 to make a custom view showing only PolicyPak GPCR events in the
application log.

![tuning_and_troubleshooting_17](/images/policypak/troubleshooting/grouppolicycompliancereporter/tuning_and_troubleshooting_17.webp)

Figure 76. Creating a custom view for PolicyPak GPCR events.

If asked by PolicyPak Support, be prepared to export these events for analysis.
