## VDI

VDI, or desktop virtualization, is defined when a full Windows 10 computer is virtualized, not when
a specific server is being used for sessions. Licensing for PolicyPak Cloud with VDI
varies depending on your purchase.

- For legacy PolicyPak Cloud monthly and PolicyPak Cloud yearly
  customers, you may consume a license with a VDI machine that has a PolicyPak Cloud
  license. Additionally, because of the Legacy Cloud Monthly or Legacy Cloud Yearly status, that
  computer will also be able to process directives via PolicyPak Group Policy Edition
  as as well, without needing an explicit, corresponding PolicyPak On-Premise license.
  (This was called an Implied On-Prem License and is only available with legacy PolicyPak Cloud licensing methods.)
- For PolicyPak Enterprise and PolicyPak Professional customers, you may
  consume a single PolicyPak Cloud license for use with a single VDI machine. Note
  that when a computer acquires a PolicyPak Cloud license, the computer will not
  process on-prem (or MDM or SCCM) directives as well, unless a corresponding license for that
  method is acquired (and deployed) as part of your purchase agreement.
- For PolicyPak SaaS customers, licenses are only available for PolicyPak Cloud, and on-prem licensing mechanisms are not available. Therefore, you may consume a
  single PolicyPak Cloud license for use with a single VDI machine.
