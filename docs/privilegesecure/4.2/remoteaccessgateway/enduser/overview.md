# End User Overview

This topic and its subtopics are written for users who have been assigned as a Remote Access
Gateway User.

New users must go through the MFA registration process before they can log in. Privilege
Secure requires a multi-factor authentication (MFA) solution (Authenticator, DUO, Symantec VIP etc)
for all user accounts. Upon initial login, the user must complete MFA registration to
proceed with using Privilege Secure. We recommend that you check with the organization's
Administrators for login requirements.
