# If I want to totally stop using PolicyPakCloud on an endpoint, how would I remove the PolicyPak Cloud client pieces remotely?

If you use Netwrix PolicyPak Cloud UI and use the **Delete
permanently** command, the next time the Cloud Client syncs to the Cloud Service all cloud pieces
(Cloud agent and Cloud CSE) are physically removed from the endpoint automatically.
