# Installation Guide

This guide is designed to help you install Usercube in a production environment.

## Target Audience

This guide is intended for **system administrators** and **system architects**.

Required knowledge includes:

- Windows Server administration
- Internet Information Services (IIS) administration
- SQL Server administration

## Overview

The installation of Usercube requires architectural decisions to be made. An
[overview](/docs/identitymanager/6.1/installation-guide/overview/index.md) of the
architecture and available configurations will help you make informed decisions.
