# GroupID Applications

Using Admin Center, you can create and manage the following GroupID applications:

- [GroupID Portal](/docs/directorymanager/11.0/signin/applications/portal/overview.md)
- [Data Service](/docs/directorymanager/11.0/signin/service/dataservice/overview.md)
- [Security Service](/docs/directorymanager/11.0/signin/service/securityservice/overview.md)
- [Mobile Service](/docs/directorymanager/11.0/signin/service/mobileservice/overview.md)

Moreover, you can manage some basic deployment and log settings for the following applications:

- [Admin Center](/docs/directorymanager/11.0/signin/applications/admincenter.md)
- [Replication Service](/docs/directorymanager/11.0/signin/service/replicationservice.md)
- [Email Service](/docs/directorymanager/11.0/signin/service/emailservice.md)
- [Scheduler Service](/docs/directorymanager/11.0/signin/service/schedulerservice.md)
