# Identity Store Connection Commands

This section covers cmdlets for establishing a connection with an identity store.

- [Connect-IdentityStore](/docs/directorymanager/11.1/managementshell/identitystoreconnection/connectidentitystore.md) – Connects to an identity store using the
  authentication mode mentioned.
- [Get-Token](/docs/directorymanager/11.1/managementshell/identitystoreconnection/gettoken.md) – Gets a token from the Directory Manager Security service.
