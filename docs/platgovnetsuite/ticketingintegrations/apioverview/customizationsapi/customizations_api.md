# Customizations API

The Customizations API provides external access to retrieve customization from a NetSuite account.

Customizations API command:

- [Get Customizations](/docs/platgovnetsuite/ticketingintegrations/apioverview/customizationsapi/get_customizations.md)

Try the [Customizations](https://documenter.getpostman.com/view/30883336/2s9YeABubu) API in Postman.
The **Get Customizations** API is published to Postman, where you can try it out and test it.
