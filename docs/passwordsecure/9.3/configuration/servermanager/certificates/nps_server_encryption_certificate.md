# Netwrix Password Secure Server Encryption Certificate

With the update to the version 8.16.0 the Netwrix Password Secure Server Encryption Certificate will
be added automatically.

![NPWS Server Encryption](/images/passwordsecure/9.2/configuration/server_manager/certificates/nps-server-encryption_1014x771.webp)

This certificate is required for offline license activation. Additional features that use this
certificate may be added in future releases.

:::warning
Export this certificate separately. Store it in a secure location outside of the server.
:::
