# How-Tos

These guides will help you configure role assignment with practical step-by-step procedures.

- #### [Restrict the Assignment](/docs/identitymanager/6.1/integration-guide/role-assignment/how-tos/restrict-assignment/index.md)
    Use filters on dimensions and/or roles to restrict the assignment of a role or resource
    type.- ####
    [Infer Single Roles with a Composite Role](/docs/identitymanager/6.1/integration-guide/role-assignment/how-tos/infer-single-roles/index.md)
    Assign several single roles via the assignment of one composite role.- ####
    [Configure Indirect Permissions ](/docs/identitymanager/6.1/integration-guide/role-assignment/how-tos/configureindirectpermissions/index.md)
    Define indirect resource rules to use indirect permissions in Usercube.
