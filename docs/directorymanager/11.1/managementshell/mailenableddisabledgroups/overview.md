# Mail-Enable/Disable Groups Commands

This section covers cmdlets for enabling and disabling groups for email.

- [Disable-DistributionGroup](/docs/directorymanager/11.1/managementshell/mailenableddisabledgroups/disabledistributiongroup.md) – disables a group's email capability.
- [Enable-DistributionGroup](/docs/directorymanager/11.1/managementshell/mailenableddisabledgroups/enabledistributiongroup.md) – enable a group's email capability.
