# Add Secret Vault

Secret Vaults can be used to store and securly provide Privilege Secure users with access to credentials and files.

Follow the steps below to add a new secret vault to the console.

**Step 1 –** Navigate to the **Policy** > **Resources** page.

**Step 2 –** Click the **+ Add** button and select New Secret Vault from the drop-down list.

![Add secrete Vault Resource](/images/privilegesecure/25.12/accessmanagement/admin/configuration/add/addsecretvault.png)

**Step 3 –** Enter the following information:

- New Secret Vault Name — Name of the new secret vault
- Description _(optional)_ —Description of the policy

**Step 4 –** Click **Save**.

Next, add credentials and/or files to the vault. See [Secrets Vault Details page](/docs/privilegesecure/25.12/admin/interface/resources/detailspages/secretvault.md). 

For information on allowing users access to the credentials and files in the vault, see [Access Policy](/docs/privilegesecure/25.12/admin/interface/accesspolicy/addaccesspolicy.md) and create an Access Policy that is Credential based. Then see [Credential-based Access Policy](/docs/privilegesecure/25.12/admin/interface/accesspolicy/credentials.md)
