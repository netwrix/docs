# How to Maintain the Workforce Directory

How to keep the workforce directory up to date.

## Overview

![Process Schema - How to Implement a New System](/images/identitymanager/user-guide/global-process/howto-maintaindirectory/globalprocess_schemamaintain.webp)

## Process Details

Be aware that the integration of an IGA tool is an iterative process. Thus, after following the [How to Start](../../user-guide/global-process/howto-start) process and creating the workforce directory, you can come back at any time and complete the directory that you started [Update Identity Data](../../user-guide/maintain/identity-data-modification).
