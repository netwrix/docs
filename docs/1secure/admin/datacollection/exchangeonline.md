# Exchange Online Auditing

Before adding an Exchange Online data source for your organization, identify the account to use
for data collection. You specify this account in the monitored item (Office 365 tenant) settings.

Netwrix 1Secure accesses the cloud-based Office 365 infrastructure using a dedicated Microsoft
Entra ID application, formerly Azure AD. A user with an administrative role must create this app
manually and assign the required permissions. See the
[App Registration and Configuration in Microsoft Entra ID](/docs/1secure/configuration/registerconfig/registerconfig.md)
topic for additional information.
