# Safari

## Installing the add-on

The Safari Add-on can be downloaded from the following link:
[Add-on Safari](https://download.passwordsafe.de/v9/Netwrix_Password_Secure-9.1.3.dmg).

To install it, double-click the downloaded file. A window opens where you then
drag and drop the Netwrix Password Secure logo onto the applications.
