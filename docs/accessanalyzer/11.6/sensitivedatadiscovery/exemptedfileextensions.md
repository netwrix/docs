# Exempted File Extensions

The following file extensions are always exempted from Sensitive Data Discovery scans:

- \*.exe
- \*.msi
- \*.bat
- \*.bin
- \*.sys
- \*.dll
- \*.iso
- \*.cab
- \*.cpl
- \*.cur
- \*.dmp
- \*.drv
- \*.lnk
- \*.scr
- \*.cdf-ms
- \*.svn-base
