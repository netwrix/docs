# Create Active Directory User Objects

In an Active Directory identity stores, you can create the following types of users:

- User – See the [Create an AD User](/docs/directorymanager/11.1/portal/user/create/AD/user.md) topic for additional information.
- Mailbox – See the [Create an AD Mailbox](/docs/directorymanager/11.1/portal/user/create/AD/mailbox.md) topic for additional information.
- Contact – See the [Create an AD Contact](/docs/directorymanager/11.1/portal/user/create/AD/contact.md) topic for additional information.
