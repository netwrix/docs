# Getting Started with Activity Sessions

Click **Settings** to begin configuring activity sessions.

:::note
After you click **Save**, activity sessions are available across your environment.
:::

## Prerequisites

:::note
You must have administrative privileges before proceeding. Only administrators can configure activity sessions.
:::

## Configure an Activity Session

Follow these steps to get started:

1. Navigate to **Admin > Activity Sessions**.
2. Click **New Session** to create a new activity session.
3. Type a name into the **Session Name** field  and select a session type.
4. Select a session type to define the scope of monitoring, manage permissions, and control access levels.
5. Session names must be unique.
6. Click **Save** to apply your changes.

## Monitor Activity

After you configure activity sessions, monitor them from the dashboard. Use the dashboard to view active sessions, review logs, and manage session settings.
