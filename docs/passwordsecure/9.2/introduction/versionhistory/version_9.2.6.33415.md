# Changes

#### Server Manager
- The legacy web app was removed from the product completely.

#### Server
- The SAML feature will soon be removed from Netwrix Password Secure! As a result of this announcement, it can no longer be installed on new systems. In addition, it can no longer be enabled for new databases.

# Fixes

#### Server
- The reports "All applications" and "Activities of the applications" are created correctly again.

#### API (.NET and JavaScript)
- When logging in with an API key, it is no longer required to enter a second factor if such is configured for the user who created the key.
