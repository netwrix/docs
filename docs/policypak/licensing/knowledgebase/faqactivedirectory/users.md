# Does LT count users?

This is what makes Netwrix PolicyPak licensing so easy.As long as
a computer is licensed for PolicyPak, all/any users logged on that computer will
receive all computer and user GPOs involving PolicyPak. This means that the users and
computers can reside in separate OUs within your Active Directory structure. Only the computer needs
to be licensed.
