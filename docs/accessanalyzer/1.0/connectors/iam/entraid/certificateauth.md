# Certificate-Based Authentication

:::note
Certificate-based authentication for Entra ID is planned for a future release. In v1.0, the Entra ID connector uses OAuth2 client credentials (client secret) authentication. See [App Registration and Permissions](/docs/accessanalyzer/1_0/connectors/iam/entraid/appregistration).
:::
