# Available Scanning Options

| Scan Option | Description | Available Configurations |
| --- | --- | --- |
| **Users, Groups, and Roles** | Scans Entra ID users, groups, and role assignments. The first scan runs in full; subsequent scans run differentially, collecting only changes since the last run. | None |
