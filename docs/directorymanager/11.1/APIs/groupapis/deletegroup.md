# Delete a Group

Use this API to delete a group from an identity store.

## Endpoint

**https://machinename:4443/GroupIDDataService/api/IdentityStores/{identitystoreId}/Groups/{GroupIdentity}**

## HTTP Method

**DELETE**

```
}
```
