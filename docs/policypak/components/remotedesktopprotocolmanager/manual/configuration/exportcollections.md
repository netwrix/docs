# Exporting Collections

Appendix A: Advanced Concepts explains how to use the PolicyPak Exporter to wrap up
any PolicyPak directives and deliver them using Microsoft Endpoint Manager (SCCM and
Intune), KACE, your own MDM service, or PolicyPak Cloud. To export a policy for later
use using PolicyPak Exporter or PolicyPak Cloud, right-click the
collection or the policy and select **Export to XML**. This will enable you to save an XML file,
which you can use later.
