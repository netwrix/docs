## Installation Steps

The software is installed by the administrator on the server. After the installation is completed, the system will be configured by opening the admin panel.

1. Log in to the admin console
2. Click the **Settings** tab
3. The checkbox(es) should be selected by you for the features you want
4. As shown below, the dropdown menu lets you select your region
5. Configure the network settings
6. Click **Save** to apply the changes

:::note
The settings were recently updated. The setup process isn't what it was before — it is streamlined.
:::
