# Process Protection Folder Templates

The **Templates** > **Microsoft** > **Process Protection** folder contains the following templates:

| Template          | Description        | TAGS |
| ----------- | ------------------- | ---- |
| Agent Hardening - Monitor | Detect attempts to terminate or suspend SIWindowsAgent.exe, SBTService.exe, and SIGPOAgent.exe process. | NEW 8.0 TEMPLATES |
| Agent Hardening - Protect | Prevent attempts to terminate or suspend SIWindowsAgent.exe, SBTService.exe, and SIGPOAgent.exe process. | NEW 8.0 TEMPLATES |
