# Netwrix Password Secure Server Encryption Certificate

With the update to version 8.16.0, the Netwrix Password Secure Server Encryption Certificate is
added automatically.

![NPS Server Encryption](/images/passwordsecure/9.2/configuration/server_manager/certificates/nps-server-encryption_1014x771.webp)

This certificate is required for activating an offline license. Additional features that depend on this certificate may be added in future releases.

RECOMMENDED: Export this certificate separately.
