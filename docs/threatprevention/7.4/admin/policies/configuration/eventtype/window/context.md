# Select Active Directory Contexts Window

The Select Active Directory Contexts window provides a list of available contexts to either include
or exclude. Selected objects are added to the filter from where you clicked the **Add** (+) button
to open this window.

![Select Active Directory Contexts Window](/images/threatprevention/7.4/admin/policies/adcontexts.webp)

- Select either Browse Mode or Search Mode.

- Select an Agent from the drop-down menu and click **Connect**.
- Navigate through the domain tree and check the desired item(s) in the **Results** pane on the
  right.
- Then click **OK** to close the window.

See the
[Selection Windows](/docs/threatprevention/7.4/admin/policies/configuration/eventtype/window/overview.md)
topic for additional information.

The selection is displayed in the appropriate box of the AD Account filter, the AD Context filter,
or the AD Objects and Containers filter.
