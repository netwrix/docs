# PolicyPak Cloud + Azure AD: Better Together for Computer ILT and Computer Policy Targeting

Netwrix PolicyPak Cloud lets you marry your PolicyPak Cloud account to Azure AD. When you do you'll turn on COMPUTER SIDE targeting for ILT and
also the ability to link policies to Azure Groups.
