# Install software with SMB (standard share)

Want to deploy software using Netwrix PolicyPak? You can do that
with PolicyPak Remote Work Delivery Manager. Just point toward the file and ... that's
it. PolicyPak will also uninstall the software when the policy no longer applies. It's
easy, and fast. (And Powerful!)
