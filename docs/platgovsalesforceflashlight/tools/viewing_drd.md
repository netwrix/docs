# Dependency Relationship Diagram

Strongpoint's Dependency Relationship Diagram (DRD) displays objects, customizations and their
relationships and dependencies including scripts, reports and workflows in a graphical diagram.

To use the entity diagram:

1. Open **Flashlight** > **Tools** > **Dependency Relationship Diagram**
2. Select the object to view.
3. The diagram will display the object and customizations under it. Use **Show/Hide Standard
   Fields** to toggle the display of Standard Fields for easier searching of non-standard fields.
4. Click on any item to expand the graphical representation of objects, dependencies and associated
   relationships.
5. Click **Open Record** to open the customization record for the item.
