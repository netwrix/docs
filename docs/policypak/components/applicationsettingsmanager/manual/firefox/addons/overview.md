# Add-Ons

Netwrix PolicyPak can manipulate Firefox add-ons by enabling and
disabling add-ons of all types. PolicyPak can also force the installation of or force
the removal of specific add-ons. To find Firefox's add-ons, select "Add-ons" within Firefox, as
shown In the figure shown.

![add_ons](/images/policypak/applicationsettings/preconfigured/firefox/addons/add_ons.webp)

The figure shown. The Add-ons tab.
