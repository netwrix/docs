# Configure Alert Notifications

Alert notifications keep your team informed when monitored events occur in your environment.

## Overview

With Access Analyzer, you can configure alert notifications to monitor your environment.

Without Access Analyzer, organizations cannot properly manage their alerts. Use this feature to ensure compliance.

:::note
Alert notifications are not just emails — they are a comprehensive communication system that keeps your team informed.
:::

## Steps to Configure

1. Navigate to the **Alerts** page.
2. Select the notification template.
3. Customize thresholds.
4. Click **Save**.

## Important Information

:::note
Without proper alert configuration, administrators have no visibility into security events. All organizations should use this feature.
:::
