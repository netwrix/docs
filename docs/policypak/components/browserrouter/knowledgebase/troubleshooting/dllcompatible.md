# PPBRAgentExeIE_01.DLL error message occurs about Internet Explorer enhanced security. What should I do?

When running IE and Netwrix PolicyPak Browser Router, the
following error could occur:

![441_1_image004](/images/policypak/troubleshooting/error/browserrouter/441_1_image004.webp)

This message is caused by IE Enhanced Security mode. To get PolicyPak Browser Router
to work properly, set IE specifically to have these two checkboxes unchecked.

![441_2_image0012](/images/policypak/troubleshooting/error/browserrouter/441_2_image0012.webp)

Then, restart IE (a reboot is not required).

Note also that if the top checkbox is checked, the error does not occur, but PolicyPak

Browser Router still will not run.
