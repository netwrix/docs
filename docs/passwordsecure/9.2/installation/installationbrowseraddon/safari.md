# Safari

## Installing the add-on

The Safari Add-on can be downloaded from the following link:
[Add-on Safari](https://releases.netwrix.com/products/passwordsecure/9.2/passwordsecure-MacApp-9.2.6.33415.dmg).

To install it, double-click the downloaded file. A window opens where you drag and drop the Netwrix Password Secure logo onto the applications.
