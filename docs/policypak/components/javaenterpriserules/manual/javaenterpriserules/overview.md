# Troubleshooting

Troubleshooting Netwrix PolicyPak Java Rules Manager is somewhat
different from troubleshooting other PolicyPak components. This is because PolicyPak
Java Rules Manager only applies to the Computer side and not to the User side. The typical problem
encountered with PolicyPak Java Rules Manager is that RIA websites don't honor the
version of Java JRE you expect on an endpoint. The sections below list the most common reasons why
they don't and provide some troubleshooting steps.
