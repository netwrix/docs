# Sensitive Data

Sensitive data configuration controls how Access Analyzer identifies and classifies sensitive content during scans. Settings apply globally across all source groups that include sensitive data scanning.

Configuration options include classification criteria, file type filters, and scanning depth. These settings determine what qualifies as sensitive data in scan results and how it is reported in the dashboards.
