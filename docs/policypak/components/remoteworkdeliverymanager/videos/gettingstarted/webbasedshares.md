# Install software using web-based shares

Want to deploy software OVER THE WEB using Netwrix PolicyPak? You
can do that with PolicyPak Remote Work Delivery Manager. Just point toward the file on
a web service like S3, Dropbox, Azure, etc. And ... that's it. PolicyPak will also
uninstall the software when the policy no longer applies. It's easy, and fast. (And Powerful!)
